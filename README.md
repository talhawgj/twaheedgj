## 🚀 Featured Projects

### 🌍 [Geodetic Conversion API & UI](https://github.com/talhawgj/Conversion-API)
**Full-stack coordinate transformation system** with millimeter-level precision
- FastAPI backend for WGS84 ↔ UTM Zone 40S conversions
- EGM2008 geoid model for ellipsoidal to orthometric height transformation
- Interactive React/Leaflet web interface with drag-and-drop CSV upload
- Bulk batch processing up to 10MB with real-time map visualization
- **Tech**: Python, FastAPI, Pyproj, React, TypeScript, Leaflet

### 🛣️ [OSRM Routing with Mapbox](https://github.com/talhawgj/Routing-OSRM)
**Self-hosted offline routing engine** for Pakistan using custom OSM data
- Custom OSRM backend compiled from OpenStreetMap data
- Sub-50ms response times with zero per-request cost
- Turn-by-turn navigation with multi-waypoint support
- Mapbox GL JS 3D terrain visualization
- **Tech**: Next.js, TypeScript, OSRM, Mapbox, Docker

### 📊 [Elevation Profile Viewer](https://github.com/talhawgj/elevation-profile)
**3D terrain analysis tool** for visualizing elevation cross-sections
- Draw transect lines on 3D Mapbox globe
- Automatic elevation sampling via Open Elevation API
- Interactive elevation charts with Recharts
- Turf.js coordinate interpolation
- **Tech**: React 19, Next.js, Mapbox, Turf.js, Recharts

### 🛰️ [Sentinel-2 Burn Scar Detection API](https://github.com/talhawgj)
**Remote sensing analysis pipeline** for automated burn area detection
- Google Earth Engine integration for satellite imagery
- NBR (Normalized Burn Ratio) calculation workflow
- Geospatial vector processing and classification
- FastAPI REST endpoints for fire monitoring
- **Tech**: Python, Google Earth Engine, FastAPI, GeoPandas
